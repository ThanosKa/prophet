# Pricing — Prophet

> AI-powered Chrome extension with Claude AI in the browser side panel and 18 browser-automation tools. Pay-per-use credits where 1 credit = 1 cent of actual API cost.

Last updated: 2026-05-04

## Plans

### Free
- Price: $0/month
- Credits included: $0.20 (one-time, does not renew)
- Models: Claude Haiku 4.5, Sonnet 4.6, Opus 4.6 (all 3 available)
- Browser automation: Yes — full 18 tools
- Rate limit: 60 chat req/min
- Card required: No
- Roughly: 50 Haiku messages OR 10 Sonnet messages

### Pro — $9.99/month
- Credits included: $11/month (+10% bonus over price)
- Models: All Claude models (Haiku 4.5, Sonnet 4.6, Opus 4.6)
- Browser automation: Yes — full 18 tools
- Rate limit: 120 chat req/min
- Persistent chat history
- Cancel anytime; credits reset monthly (no rollover)

### Premium — $29.99/month
- Credits included: $35/month (+17% bonus over price)
- Models: All Claude models
- Browser automation: Yes — full 18 tools
- Rate limit: 240 chat req/min
- Persistent chat history
- Cancel anytime; credits reset monthly (no rollover)

### Ultra — $59.99/month
- Credits included: $70/month (+17% bonus over price)
- Models: All Claude models
- Browser automation: Yes — full 18 tools
- Rate limit: 240 chat req/min
- Persistent chat history
- Cancel anytime; credits reset monthly (no rollover)

## Key facts
- Credit conversion: 1 credit = $0.01 of actual Claude API cost
- All sales final; no refunds
- No hidden fees, no per-seat pricing, no usage overages — when credits hit zero, requests stop
